## Featured Projects

### [Fureal - Smart AI Chatbot](https://github.com/kanishkapandey23/Fureal)
A multilingual AI chatbot that detects user emotion (happy, sad, angry, confused) and responds empathetically.
- Hugging Face models for emotion classification, paired with the Gemini API for empathetic response generation
- RAG pipeline built on a Pinecone vector database to ground responses in relevant document context
- React.js + Tailwind CSS + Vite frontend, backed by a FastAPI service

`React` `Tailwind CSS` `Vite` `FastAPI` `Hugging Face` `Gemini API` `Pinecone`

### [Real-Time Chat Application](https://github.com/kanishkapandey23)
A low-latency, room-based chat platform for isolated multi-user conversations.
- Built on WebSockets and Socket.IO for real-time, bidirectional communication
- Room-based architecture so multiple groups can chat in isolated channels
- Manages connection, disconnection, and live-update state across concurrent users

`WebSockets` `Socket.IO` `Node.js`

### [Deep Researcher Agent](https://github.com/kanishkapandey23/Deep-Researcher-Agent)
An agent-driven research assistant that automates the process of gathering and synthesizing information.

### [Hand Gesture Recognition](https://github.com/kanishkapandey23/Hand-gesture-Recognition)
A computer-vision project that classifies hand gestures in real time.

`Python` `OpenCV` `Jupyter Notebook`
